# claude-ichiba (市場)

A [Claude Code](https://claude.ai/code) [plugin marketplace](https://code.claude.com/docs/en/plugin-marketplaces) by Gwenneg Lepage.

## Plugins

| Plugin | Description |
|---|---|
| [togi](https://github.com/gwenneg/togi) | Captures AI coding friction and turns recurring doc gaps into context-doc pull requests |

## Install

In Claude Code:

```
/plugin marketplace add gwenneg/claude-ichiba
/plugin install togi@claude-ichiba
/reload-plugins
```
